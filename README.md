# PInvokeFolderOpener

Created in 2021/4

## What is this?

Reimplement the folder selection dialog for study of CsWin32.

### Memo

CsWin32 automatically adds `AllowUnsafeBlocks = true`, so it shouldn't be necessary. However, I defined it implicitly just in case.
